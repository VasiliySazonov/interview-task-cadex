# interview-task-cadex

The project was built and tested on **Ubuntu 22.04** kernel: **5.19.0** <br>
using **gcc 11.3.0**, **cmake 3.22.1** and **make 4.3**

## Building

`git clone https://github.com/VasiliySazonov/interview-task-cadex.git `

`mkdir build`

`cd build`

`cmake ..`

`make`

## Running
`./main`
